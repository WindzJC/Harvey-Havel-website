# Harvey Havel website asset pack

Files collected for the Harvey Havel author website build.

## Included assets
- `author-headshot-amazon.jpg` - author headshot from Amazon author/profile image.
- `author-photo-from-manuscript.png` - color author image extracted from the uploaded manuscript PDF.
- `the-queen-of-intelligence-cover.jpg` - cover for *The Queen of Intelligence: A 9/11 Conspiracy Novel*.
- `noble-mccloud-cover.jpg` - cover for *Noble McCloud: A Novel*.
- `the-wild-gypsy-of-arbor-hill-cover.jpg` - cover for *The Wild Gypsy of Arbor Hill*.
- `stories-from-the-fall-of-the-empire-cover.jpg` - cover for *Stories from the Fall of the Empire*.
- `the-odd-and-the-strange-cover.jpg` - cover for *The Odd and The Strange: A Collection of Very Short Fiction*.
- `mother-a-memoir-cover.jpg` - cover for *Mother: A Memoir*.

## Suggested usage
- Hero/about section: `author-headshot-amazon.jpg`
- Optional secondary/about image: `author-photo-from-manuscript.png`
- Featured book: `the-queen-of-intelligence-cover.jpg`
- Library/grid books: the rest of the cover files

## Notes
- Several covers were only available from public retailer/catalog thumbnails, so some files are low resolution.
- If you want, these should be treated as starter web assets until higher-resolution publisher files are available.
